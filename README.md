# Groupe1_Open
Création de l'application pour le Module OPEN 2026, une application de traduction/révision qui permet
de choisir sa langue, le sens de traduction, un mode aléatoire ou un mode personnalisé qui accentue sur les mots
qui sont les moins réussi et par catégorie également.

Cette application laisse aussi la possibilté à l'utilisateur
de fournir de nouveaux mots à la base de donnée de l'appli en renseignant sa langue qu'elle soit nouvelle pour
l'application ou non, de même avec les catégorie. 

Nous avons donc diviser l'application en 4 onglets, la présentation et 
les crédits respectivement au début et en fin de l'interface, avec l'onglet apprentissage en second qui concentre les fonctionnalités
de traduction et révision, et enfin l'onglet paramétrage ou les ajouts dans la base de données sont possibles.
